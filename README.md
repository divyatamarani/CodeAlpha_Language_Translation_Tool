# 🌍 AI Language Translation Tool

A professional AI-powered Language Translation Tool built using Python and Streamlit.

## 🚀 Features

- Translate between multiple languages
- Modern and clean UI
- Copy translated text
- Translation history
- Fast translation using Google Translator

## 🛠 Technologies Used

- Python
- Streamlit
- Deep Translator
- Pyperclip

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
